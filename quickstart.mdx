LumeGates is fully compatible with the OpenAI API. Simply update the `API Key` and the `Base URL` in your existing code.

## 1. Get Your API Key

1. Log in to the [LumeGates Dashboard](https://lumegates.com).
2. Navigate to the **Top-up** section to add credits.
3. Go to the **Tokens** page and click **Create Token**.

## 2. Configure the Base URL

All requests should be sent to: `https://api.lumegates.com/v1`

## 3. Make Your First Request

<CodeGroup>

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_LUMEGATES_TOKEN",
    base_url="https://api.lumegates.com/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello LumeGates!"}]
)
print(response.choices[0].message.content)
```

```bash cURL
curl https://api.lumegates.com/v1/chat/completions \
  -H "Authorization: Bearer YOUR_LUMEGATES_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "Hello LumeGates!"}]
  }'
```

```javascript Node.js
import OpenAI from "openai";

const openai = new OpenAI({
  apiKey: "YOUR_LUMEGATES_TOKEN",
  baseURL: "https://api.lumegates.com/v1"
});

const response = await openai.chat.completions.create({
  model: "gpt-4o",
  messages: [{ role: "user", content: "Hello LumeGates!" }],
});
```
